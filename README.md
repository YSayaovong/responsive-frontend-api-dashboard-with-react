# Responsive Frontend API Dashboard (React)

This project is a responsive data dashboard built with React.  
It highlights frontend engineering skills through UI architecture, API integration, and adaptive layouts across desktop, tablet, and mobile.

## Overview

This application demonstrates the ability to build a modern React-based dashboard interface that consumes live API data. The focus is entirely on the frontend: rendering data, designing UI components, and ensuring responsiveness on all screen sizes.

The project is intended to showcase capabilities relevant to a **Frontend Engineer / React Developer** role.

---

## Tech Stack

- **React** (Create React App)
- **JavaScript (ES6+)**
- **CSS** (Flexbox, responsive layout)
- **REST APIs**
- **Vercel Deployment**

---

## Features

- **Responsive UI** designed to work on mobile, tablet, and desktop.
- **API-driven components** that fetch and display live data.
- **Functional components with React Hooks** for state and effects.
- **Reusable layout and card components.**
- **Clean project structure** following common React best practices.

## Getting Started

### 1. Clone Repository
```bash
git clone https://github.com/YSayaovong/responsive-frontend-api-dashboard-with-react.git
cd responsive-frontend-api-dashboard-with-react
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm start
```

Your app runs on:
```
http://localhost:3000
```

### 4. Build for Production
```bash
npm run build
```

Creates an optimized `build` directory ready for deployment.
